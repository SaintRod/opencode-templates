# Purpose

The purpose of this document is to create an in-depth highly-detailed **plan** of action for how this project will be completed.
The plan consists of two sections:

- Discussion
- Tasks

Within each section there are phases, numbered 1-n.
We will iterate on the plan until I'm satisfied.

In the Discussion section we will discuss how to fulfill the mission in words by phase.
Be succinct, clear, and explicit (e.g., what functions are needed, what are the args and types of these functions, etc.)
Describe every step that's needed and don't be ambigious.

In the Task section we will create a list of tasks in logical or sequential order by phase.
These tasks will act as our kanban board, describing what needs to be done, in what order, and what has been completed.
As you complete a task mark it complete in this file.
